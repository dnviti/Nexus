CSS Classes Available

The following CSS utility classes are available for use in your plugins:

/* Layout */ .container, .flex, .grid .items-center, .justify-between, .gap-2, .gap-4 /* Buttons */ .btn, .btn-primary, .btn-secondary, .btn-action /* Forms */ .form-group, .form-label, .form-control, .form-check /* Cards */ .card, .card-header, .card-body, .card-elevated /* Status */ .status-indicator, .status-online, .status-offline, .status-warning /* Data Display */ .table, .stats-grid, .stat-card, .stat-value /* Progress */ .progress, .progress-bar /* Utilities */ .text-center, .text-left, .text-right .mt-1, .mt-2, .mb-1, .mb-2 .hidden, .loading
Plugin Integration

To use these components in your plugin, simply include the CSS classes in your HTML templates or use the Python component classes:

# Python component usage from nexus.ui.components import Button, Card, Alert # Create components programmatically button = Button("Click Me", variant="primary") card = Card(title="My Card", elevated=True) alert = Alert("Success message", alert_type="success") # Render to HTML html = button.render()
